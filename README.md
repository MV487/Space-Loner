#  Space Loner

***It is a 3D, multiplayer, strategy, survival game***


## setup
### need python 3.7
```bash
git clone --recursive https://github.com/MV487/Space-Loner.git
python -m venv env
pip install -r requirements.txt
cd RenderPipline && python.exe setup.py
```
